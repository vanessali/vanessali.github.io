Arduino Visualizer

An Rdio hackday project developed with fellow colleague, Bryan Hughes. Using the Meyda javascript audio framework for real-time feature extraction, we collected power spectrum of 5 frequency bands and mapped it onto a Neopixel grid using Arduino. The system is comprised of the client from the Rdio player, the server for processing raw FFT data, and the Arduino firmware. Github repo here.
